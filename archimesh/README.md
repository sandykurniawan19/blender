Archimesh
=======

Blender add-on for creating architecture elements.

- Rooms
- Houses
- Columns
- Stairs
- Doors
- Windows
- Tile roofs
- Kitchen cabinets
- Japanese curtains
- Roller curtains
- Venetian blind
- Books
- Lamps

You can get more information in: https://www.youtube.com/playlist?list=PLQAfj95MdhTJ7zifNb5ab-n-TI0GmKwWQ
